"Quick Docs"
-An Online Document Editor with Spell checker and grammar Correction
-Using Django (Python Framework)

Detailed Explanation

Demo Video

“Online Document Editor with Spell Check and Grammar Detection” is a web application based project. To write a document we may need a software to install (for example, Microsoft office word), but a system may not have a document editor installed. In that scenario, an online web-based document editor is preferable which we don't need to install and we can do all the task within the web like text formatting, creating tables, adding multimedia files etc. Moreover, a system may not be compatible to install and use particular document editor software. As a result, it saves a lot of time and resources.


To design this project, we have used Django framework, a python framework. To provide all the important features, we have used a lot of methodologies. To create the text editor, summernote is used which is an open-source rich-text-editor. For grammar and spell correction, “Beyond Grammar” by proWriting has been used. To build the chatbot for personal assistance, Google's dialogflow is used. To design a beautiful and responsive UI, bootstrap is used.


The web application is very much responsive and it offers a beautiful dashboard that attracts the users. It offers a text editor where a user can write his document and save it to his account and it can be edited later. It is able to correct the grammars and spells if required. The chatbot for personal assistance is working fine and it can answer the queries asked. The security measures in this application are very strong. We have named the system "Quick Docs".


Features:

  • ✅The system offers a full fledged dashboard.
  • ✅The system includes a chatbot for assistance.
  • ✅The system is able to check common spellings and suggest correct spellings.
  • ✅The system is aslo able to correct the grammar if required.
  • ✅The document editor includes various text formatting features and is also able to import other multimedia files and links within it.
  • ✅The document editor offers to categorise the documents accordingly to the requirement of the user.
  • ✅The privacy of the documents can be set by the user. By default it is set to private.
  • ✅A document can be edited and can be downloaded in “pdf” format for the user.
  • ✅The documents saved can be liked by the viewers.

Tools and Technologies used

  • ▶️Frameworks: Django
  • ▶️Bootstrap 5.1
  • ▶️Summernote (Rich Text Editor)
  • ▶️ProWriting’s “Beyond Grammar”
  • ▶️Google dialogflow (Chatbot- For Perosnal Assistance)
