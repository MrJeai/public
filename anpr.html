Automatic Number Plate Recognition (ANPR) System
-Using Deep Learning

Detailed Explanation

Demo Video

Automatic Number Plate Recognition (ANPR) is a technology that uses optical character recognition on images to read vehicle registration plates to create vehicle location data.


Technologies towards smart vehicles, smart cities and intelligent transportation systems continue to transform many facets of human life. As a consequence, technologies such as automatic number plate recognition have become part of our everyday activities. ANPR also contributes to various use cases while eliminating human intervention. Hence, ANPR system will be an immense help to vehicle owners as well as concerned authorities for vehicle surveillance.


Features:

  • ✅This model is capable of scanning and detecting vehicle's number plates in realtime.
  • ✅A database system for keeping records of the vehicle number plates.

Tools and Technologies used

  • ▶️Programming Language used: Python
  • ▶️Deep Learning architecture used: SSD (Single Shot Detector) MobileNet V2
  • ▶️Tensorflow
  • ▶️OpenCV
  • ▶️EasyOCR
