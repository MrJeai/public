"AVA" -An android chatbot application
-Using Android Studio and Machine Learning

Detailed Explanation

Demo Video

"AVA" is an artificial intelligence (AI) powered android application that can simulate a conversation (or a chat) with a user in natural language. A chatbot is often described as one of the most advanced and promising expressions of interaction between humans and machines. Chatbot applications streamline interactions between people and services, enhancing customer experience. At the same time, they offer companies new opportunities to improve the customers engagement process and operational efficiency by reducing the typical cost of customer service


This application has been created from scratch using machine learning algorithms to create the API and android studio for the front-end.


Features:

  • ✅This application is capable of answering your queries.
  • ✅Modern UI with animations
  • ✅TTS (Text-to-speech) feature
  • ✅STT (Speech-to-Text) feature

Tools and Technologies used

  • ▶️Programming Language used: Python, JAVA
  • ▶️Machine Learning
  • ▶️Android Studio
  • ▶️Google's TTS (Text-to-speech)
  • ▶️Google's STT (Speech-to-Text)
  • ▶️Heroku server (for hosting)
